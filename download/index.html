Download Nevacord

Download Nevacord and run it. It should list all Discord installs on your System. Pick the one you would like to patch and press Install.

Download Nevacord Download NevacordCli

If the installer does not open for you, download NevacordCli instead. Run it and a terminal window should open. Just follow the instructions in the terminal.

Do not run the installer as an Administrator!

If you get a warning that the app cannot be opened, click "Run Anyways". You may need to click "more info" to see this option.

This warning is entirely harmless and only shows because the app is not signed. Signing it would cost us upwards of 300€/year.
